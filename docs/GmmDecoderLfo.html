Class: GmmDecoderLfo

GmmDecoderLfo

Lfo class loading GMM models created by the xmm library to process an input stream of vectors (models must have been trained from the same input stream). As the results of the classification / regression are more complex than a simple vector, a callback function can be passed to the constructor to handle them, or they can alternatively be queried via the readonly filterResults property.
